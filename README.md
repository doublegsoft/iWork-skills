iWork-skills
============

### iw-codemerge

代码合并，从文件到文件的替换，文件中局部方法的合并与替换。

### iw-html2image

html文件转换成图片。

### iw-funcomplex

函数复杂度计算。


```bash

iw-funcomplexity -c <完整代码>
```

### iw-funflow

函数的逻辑流程，包括Control Flow Graph和Data Flow Graph。

### iw-sourcewrite

解析markdown协议，解析出满足一下协议格式的内容就执行。协议如下：

```
filepath: <文件路径>

<完整文件代码>
```

### iw-probuild

项目编译，支持maven、cargo、cmake等等。
